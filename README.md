# Personal Portfolio Website

A modern, responsive, and professional personal portfolio website showcasing skills, projects, education, and achievements. Built with clean design principles and smooth animations for an optimal user experience.

## Project Description

This is a fully responsive personal portfolio website that includes:
- **Home/Introduction** - Professional introduction with name, tagline, and social media links
- **About Me** - Education background, skills, and career goals
- **Projects** - Showcase of projects with descriptions and technologies used
- **Contact** - Contact form and contact information (Email, Phone, Location)
- **Download CV** - Direct download link for CV in PDF format

The website features smooth animations, modern UI/UX design, and is fully optimized for all devices including mobile, tablet, and desktop.

## Technologies Used

- **HTML5** - Semantic markup and structure
- **CSS3** - Modern styling with:
  - CSS Grid and Flexbox for layouts
  - CSS Custom Properties (Variables)
  - CSS Animations and Transitions
  - Responsive Design with Media Queries
  - Backdrop Filters for glassmorphism effects
- **JavaScript (ES6+)** - Interactive features including:
  - Smooth scrolling navigation
  - Intersection Observer API for scroll animations
  - Throttled scroll events for performance
  - Mobile menu toggle
  - Form handling
  - Parallax effects
- **Font Awesome** - Icon library for social media and UI icons
- **Google Fonts (Poppins)** - Modern typography

## Installation & Setup

1. Clone the repository:
```bash
git clone https://github.com/Renabid/Portfolio.git
cd Portfolio
```

2. Open `index.html` in your web browser or use a local server:
```bash
# Using Python
python -m http.server 8000

# Using Node.js (http-server)
npx http-server

# Using VS Code Live Server extension
# Right-click on index.html and select "Open with Live Server"
```

3. The website will be accessible at `http://localhost:8000` (or the port specified)
